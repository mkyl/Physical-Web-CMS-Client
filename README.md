Physical Web CMS Client
========================

Android app to be used by visitors to view beacon content configured through the [Content Manager](https://github.com/mkyl/Physical-Web-CMS-Android). When placed in range of beacons configured with app, it will load the active exhibit and display the contents of the nearest beacon, updating as the visitor moves around the space.
